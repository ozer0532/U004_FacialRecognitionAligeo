!!! README !!!

Prasyarat:
Python 3 & library:
- OpenCV2
- SciPy
- ImageIO
- Matplotlib
- PyQt5

Cara Run:
- Jalankan terminal
- Masuk ke dalam folder bin
- Jalankan: python app.py atau python3 app.py (Linux)

Cara Menggunakan:
- Generate basis data dengan menjalankan aksi 'Generate Database' (folder hanya boleh berisi gambar *.jpg)
- Tes gambar dengan menjalankan aksi 'Test File' (pilih file *.jpg, pilih folder yang berisi database sebelumnya [TIDAK BOLEH TERMODIFIKASI!!!])

Jika Error:
- Pastikan gambar yang dibaca memiliki MUKA JELAS
- Pastikan isi folder yang dibuka tidak mengandung isi lain selain gambar *.jpg
